Week 8 – Tkinter Applications

Learning Outcome:

Learned how to create graphical user interfaces using Tkinter.

Activity Description:

Created programs that used Tkinter to build simple applications, including a temperature converter and a shopping total calculator.


Skills Demonstrated:

* Tkinter
* Dictionaries
* Functions
* User input

Reflection:

The most challenging part was connecting buttons and functions in Tkinter. I learned how to create simple GUI applications in Python.
