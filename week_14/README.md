Week 14 – Stock Management System

Learning Outcome:

Learned how to develop a larger object-oriented programming project.

Activity Description:

Created a stock management system that allowed products to be added, updated, searched, sold, and tracked.


Skills Demonstrated:

* Classes
* Objects
* Lists
* Dictionaries
* Menu systems

Reflection:

The most challenging part was connecting all the methods together. I learned how object-oriented programming can be used to create larger applications.
