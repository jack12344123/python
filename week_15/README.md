Week 15 – Helpdesk Ticket System

Learning Outcome:

Learned how to create a larger object-oriented programming application.

Activity Description:

Created a helpdesk ticket system that managed support requests, ticket statuses, password resets, and ticket statistics.

Skills Demonstrated:

* Classes
* Objects
* Methods
* Class variables
* Object-oriented programming

Reflection:

The most challenging part was managing ticket statuses and statistics. I learned how object-oriented programming can be used to solve real-world problems.
