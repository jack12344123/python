Week 7 – Lists and Introduction to Tkinter

Learning Outcome:

Learned how to use lists and create basic graphical user interfaces with Tkinter.

Activity Description:

Created programs that managed shopping lists, calculated averages using functions, and built a simple Tkinter window.


Skills Demonstrated:

* Lists
* Functions
* Tkinter
* Loops

Reflection:

The most challenging part was learning how Tkinter works. I learned how to create simple windows and use lists to store data.
